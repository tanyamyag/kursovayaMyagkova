# Проект «Чистота» (Vue + Vite)

Учебный проект по макету из Figma. Стек: **Vue 3**, **Vue Router**, **Vite**. #стек - это набор инструментов для создания продукта

## Страницы

- `/` — Главная
- `/about` — О компании
- `/pricing` — Стоимость
- `/register` — Регистрация
- `/login` — Вход
- `/employees` — Список сотрудников
- `/employees/:id` — Карточка сотрудника с отзывами и сертификатом
- `/privacy-policy` — Политика конфиденциальности
- `/faq` — FAQ (часто задаваемые вопросы)

## Как запустить

```bash
npm install
npm run dev
```

Откройте адрес из консоли (по умолчанию `http://localhost:5173`).
